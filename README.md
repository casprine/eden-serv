# eden-serv

Routes

- api/v1/featured/ -> handpicked topics
- api/v1/books -> all books
- api/v1/books?topic=\${topic} -> topic based
- api/v1/authors -> all authors
- api/v1/topics -> all topics

## Data Params

- Name
- Description
- Type {Paid or Free}
- Book Image
- Topic {dropdown}
- Author/s
  - Name
  - Other things we get ie. links like images and stuff
